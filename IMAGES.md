# Add-on Images

This add-on requires two image files:

## icon.png
- Size: 108x108 pixels
- Format: PNG with transparency
- Description: Square icon showing Claude Code logo or robot symbol
- Used in: Home Assistant add-on store listing

## logo.png
- Size: 128x128 pixels (or larger)
- Format: PNG with transparency
- Description: Logo for the add-on, similar to icon but can be more detailed
- Used in: Add-on details page header

## Creating Images

You can create these images using:
1. DALL-E or other AI image generator
2. Design tool (Figma, Canva, GIMP, etc.)
3. Export from existing Claude Code branding

### Suggested Design
- Robot or AI agent symbol
- Home Assistant color scheme integration (#41BDF5 blue)
- Clean, simple design that works at small sizes
- Transparent background

### Temporary Placeholder
Until proper images are created, you can use a simple colored square or text-based placeholder.
